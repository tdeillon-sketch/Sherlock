A = Type 1 (Le Perfectionniste) · B = Type 2 (L'Altruiste) · C = Type 3 (Le Battant) · D = Type 4 (Le Romantique) · E = Type 5 (L'Observateur) · F = Type 6 (Le Loyaliste) · G = Type 7 (L'Épicurien) · H = Type 8 (Le Chef) · I = Type 9 (Le Médiateur)

## NOTE IMPORTANTE

Ce quiz est un point de départ, pas un diagnostic définitif. Comme le rappelle le livre : "la carte n'est pas le territoire". Si deux ou trois lettres sont à égalité, lisez les descriptions complètes de ces types pour identifier celui dont la peur fondamentale et la motivation profonde résonnent le plus. C'est la motivation inconsciente — le *pourquoi* — qui détermine votre type, pas seulement vos comportements visibles.

Les deuxième et troisième scores peuvent également révéler vos **ailes** (types voisins qui influencent votre profil) ou vos **flèches** (types que vous empruntez sous stress ou en sécurité).
